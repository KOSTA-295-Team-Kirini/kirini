자주 묻는 질문 (FAQ)

KIRINI 서비스 이용 중 자주 묻는 질문들에 대한 답변입니다.

회원가입은 어떻게 하나요?
KIRINI 회원가입은 간단합니다. 상단 메뉴에서 '로그인' 버튼을 클릭한 후 로그인 페이지 하단의 '회원가입' 링크를 클릭하세요. 이메일, 비밀번호, 닉네임 등 기본 정보를 입력하면 가입이 완료됩니다. 가입 후 이메일 인증을 통해 모든 서비스를 이용하실 수 있습니다.
비밀번호를 잊어버렸어요. 어떻게 해야 하나요?
로그인 페이지 하단의 '비밀번호 찾기' 링크를 클릭하세요. 가입 시 사용한 이메일을 입력하시면 비밀번호 재설정 링크가 포함된 이메일을 보내드립니다. 이메일을 받지 못하셨다면 스팸 폴더를 확인하시거나 1:1 문의를 통해 문의해주세요.
게시글이나 댓글을 수정/삭제하고 싶어요.
본인이 작성한 게시글이나 댓글은 언제든지 수정하거나 삭제할 수 있습니다. 게시글 하단이나 댓글 우측에 있는 '수정' 또는 '삭제' 버튼을 클릭하세요. 삭제된 게시글은 복구가 불가능하니 신중하게 결정해주세요.
키보드 정보에 오류가 있는 것 같아요.
키보드 정보 페이지에서 오류를 발견하셨다면, 해당 키보드 정보 하단의 '오류 신고' 버튼을 클릭하거나 1:1 문의를 통해 알려주세요. 정확한 정보로 수정될 수 있도록 최대한 자세한 내용을 알려주시면 큰 도움이 됩니다.
새로운 키보드 용어나 정보를 추가하고 싶어요.
용어집 또는 키보드 정보 페이지 하단에 있는 '용어 추가 요청' 또는 '키보드 정보 추가 요청' 버튼을 클릭하여 신청해주세요. 관리자 검토 후 적절한 내용이라면 추가됩니다. 정확하고 유용한 정보 공유에 감사드립니다.
부적절한 게시물을 발견했어요. 어떻게 신고하나요?
모든 게시물과 댓글 옆에는 '신고' 버튼이 있습니다. 이 버튼을 클릭하여 부적절한 내용에 대한 신고를 할 수 있습니다. 신고된 내용은 관리자가 검토 후 커뮤니티 가이드라인에 위배되는 경우 조치를 취합니다.

이용안내

KIRINI 서비스 이용에 대한 안내입니다.

회원가입 및 로그인
KIRINI에 가입하시면 다양한 커뮤니티 활동과 개인화된 서비스를 이용하실 수 있습니다.
  1. 상단 메뉴에서 '로그인' 버튼을 클릭합니다.
  2. 로그인 페이지 하단의 '회원가입' 링크를 클릭합니다.
  3. 필수 정보를 입력하고 '가입하기'를 클릭합니다.
  4. 입력한 이메일로 인증 메일이 발송됩니다.
  5. 이메일 인증 후 로그인하여 서비스를 이용하실 수 있습니다.
게시판 이용하기
KIRINI의 게시판은 키보드에 관한 다양한 정보와 소식을 공유하는 공간입니다.
  1. 상단 메뉴에서 '소식/게시판'을 클릭합니다.
  2. 원하는 게시판 카테고리를 선택합니다.
  3. 게시글을 읽거나, '글쓰기' 버튼을 클릭하여 새 게시글을 작성합니다.
  4. 댓글을 남기거나 좋아요를 클릭하여 소통할 수 있습니다.
  5. 자신이 작성한 게시글과 댓글은 '마이페이지'에서 관리할 수 있습니다.
키보드 정보 찾기
KIRINI에서는 다양한 키보드 정보와 용어를 확인할 수 있습니다.
  1. 상단 메뉴에서 '키보드정보' 또는 '용어집'을 클릭합니다.
  2. 검색창을 이용하여 원하는 키보드나 용어를 검색합니다.
  3. 필터 기능을 활용하여 브랜드, 타입, 가격대 등으로 분류된 정보를 확인합니다.
  4. 키보드 상세 페이지에서 스펙, 리뷰, 관련 정보를 확인할 수 있습니다.
  5. 용어를 클릭하면 해당 용어에 대한 자세한 설명을 볼 수 있습니다.
Q&A 이용하기
키보드에 관한 궁금한 점은 Q&A 게시판에서 질문하고 답변을 받을 수 있습니다.
  1. 상단 메뉴에서 'Q&A'를 클릭합니다.
  2. '질문하기' 버튼을 클릭하여 새 질문을 작성합니다.
  3. 질문 제목과 내용을 작성하고 관련 태그를 선택합니다.
  4. 다른 사용자의 질문에 답변을 작성할 수도 있습니다.
  5. 유용한 답변에는 '도움됨' 버튼을 클릭하여 평가할 수 있습니다.
