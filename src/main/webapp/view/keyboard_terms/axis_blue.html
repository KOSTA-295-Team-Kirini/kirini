청축 (Blue Switch)

클릭 타입의 뚜렷한 클릭음과 구분감을 가진 스위치

목차
  • 1. 개요
  • 2. 특징
  • 3. 용도 및 예시
  • 4. 관련 용어

1. 개요

청축은 기계식 키보드 스위치의 한 종류로, 영어로는 Blue Switch라고 부릅니다. 클릭(Click) 타입으로, 누를 때 뚜렷한 '딸깍' 소리와 함께 확실한 구분감이 느껴지는 것이 특징입니다.

2. 특징

  • 클릭 타입: 누를 때 명확한 '딸깍' 소리와 구분감이 있음
  • 소음이 큰 편이며, 키압은 약 50g~60g 정도
  • 타이핑 시 정확한 키 입력 확인 가능
  • 타건감이 뚜렷하여 타이핑 중심 사용자에게 인기
[청축 스위치 예시 이미지]

3. 용도 및 예시

  • 타이핑을 많이 하는 사용자에게 적합
  • 소음이 문제되지 않는 환경에서 사용 권장
  • 정확한 타건감을 중시하는 사용자에게 추천
  • 게임보다는 문서 작업이나 프로그래밍 작업에 적합

4. 관련 용어

축, 적축, 갈축, 흑축
